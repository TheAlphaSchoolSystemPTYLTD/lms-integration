**getTimetables**
----
  Returns an array of structured details of timetables in JSON format.
  
* **Version History:**

  TASS v52.5 - Method Added

* **Version:**

  3

* **Permission:**

  Timetables > Timetable Setup > Timetable Definitions > View

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**

   `year_num [string]` -  Year Number

   `period [string]` -  Period Code
   
   **Optional:**
 
   none

   **Conditional:**
 
   none
